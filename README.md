SIMS CAFE Management System is a robust, cross-platform (Windows & Mobile) point-of-sale (POS) and cafe management application built with Flutter. 
Designed for seamless cafe operations, it features an offline-first architecture using SQLite with real-time Firebase cloud synchronization.

The application provides a comprehensive suite of tools for restaurant management, including:

-Order & Menu Management: Intuitive dashboard for handling orders and local menu repositories.
-Staff Management: Built-in delivery boy and staff management systems.
-LAN/Wi-Fi Device Sync: Custom WebSocket and HTTP server integration to sync multiple local devices in real-time without relying on external internet.
-Hardware Integration: Direct support for ESC/POS thermal receipt printers and the Windows Raw Printing API.
-Reporting & Analytics: Generate receipts, export data to Excel, and create detailed PDF reports.
-Auto-Updating: Built-in mobile and desktop auto-update mechanisms to keep client apps seamlessly up to date.

Built by SIMS AI.
